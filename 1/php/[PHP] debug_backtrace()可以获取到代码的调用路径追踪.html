    • [PHP] debug_backtrace()可以获取到代码的调用路径追踪
    原文出处:本文由博客园博主陶士涵提供。
    原文连接:https://www.cnblogs.com/taoshihan/p/10769944.html

    查看代码的时候,看到有使用这个函数,测试一下

    1.debug_backtrace()可以获取到代码的调用追踪,以数组形式返回

    2.debug_print_backtrace() — 打印一条回溯,直接打印出来

    <?php
class Test{
        public function __construct(){

                $ts=debug_backtrace();
                var_dump($ts);
                debug_print_backtrace();
        }   
        public function load($class){
                echo $class;
        }   
            
}
function test(){

        new Test();
}
test();
