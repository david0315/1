    • 安装rabbitMQ的PHP扩展
    原文出处:本文由博客园博主明月青天提供。
    原文连接:https://www.cnblogs.com/kenny88/p/11857645.html

    1.环境准备：centos 7.6+PHP7.3

    2.安装rabbitmq-c
    tar xf rabbitmq-c-0.9.0.tar.gz
    cd rabbitmq-c-0.9.0
    mkdir build && cd build
    cmake -DCMAKE_INSTALL_PREFIX=/usr/local/rabbitmq-c ..
    cmake --build . --target install
    cd /usr/local/rabbitmq-c/
    ln -s lib64 lib

    3.安装amqp扩展
    tar xf amqp-1.9.4.tgz
    cd amqp-1.9.4
    phpize
    ./configure --with-php-config=/usr/local/php/bin/php-config --with-amqp --with-librabbitmq-dir=/usr/local/rabbitmq-c
    make && make install
    vi /usr/local/php/lib/php.ini
    extension = amqp.so

    4.重启php-fpm
    systemctl restart php-fpm

    5.检查确认
    php -i |grep amqp
    php -m
