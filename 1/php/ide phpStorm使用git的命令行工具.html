    ide phpStorm使用git的命令行工具

    1.点击phpStorm左下角，点击terminal

    2.发现是windows自带的cmd.exe，可以将其改为git的sh.exe

    3.打开设置(File -> Settings -> Tools -> Terminal)
    4.如下图改为 git根目录下/bin/sh.exe最后加 login -i（注意双引号）

    5.重启phpStorm
