    直接进入主题

    1.首先获取git clone项目

    2.创建镜像：docker build -t="docker" .(注意千万不要忘了.)

    3.列出镜像：docker images

    4.添加端口映射：docker run --name mydocker -d -p 6100:80 容器/tag

    5.访问如：127.0.0.1:6100 报错

    报错原因：因为没有启动php-fpm和ningx

    解决方法一：1.进入容器：docker exec -it 容器  /bin/bash

    执行：Source /etc/kickStart.d/start.sh 启动php-fpm和ningx，如果不确定是否启动
