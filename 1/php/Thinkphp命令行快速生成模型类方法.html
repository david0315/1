    • Thinkphp命令行快速生成模型类方法
    原文出处:本文由博客园博主IDFC提供。
    原文连接:https://www.cnblogs.com/connectionworld/p/10795030.html

    进去cmd，切换到项目根目录，也就是think文件所在目录，执行下面的指令可以生成index模块的blog模型类文件：

    >php think make:model index/Blog

     

    生成的模型类文件如下

    <?php

    namespace app\index\model;

    use think\Model;

    class Blog extends Model{

    　　//...

    }
