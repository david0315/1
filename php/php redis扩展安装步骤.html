    • php redis扩展安装步骤
    原文出处:本文由博客园博主petter7226提供。
    原文连接:https://www.cnblogs.com/petter7226/p/11996292.html

    因为redis不是php技术自带的技术，因此我们如果要通过php程序来操作redis，需要redis设计者提供对应的操作接口（函数类）我们使用phpredis.tar.gz文件在源码编译生成一个redis.so文件

    安装步骤：

     1、下载phpredis.tar.gz

      http://redis.io/clients

    https://github.com/phpredis/phpredis

    2、上传phpredis.zip到服务器上并解压，然后进入到phpredis-develop

    3、/usr/local/php/bin/phpize  [生成 configure]

    4、./configure--with-php-config=/usr/local/php/bin/php-config

     5、make && make install

    安装成功之后最后有个路径提示

    /usr/local/php/lib/php/extensions/no-debug-non-zts-20090626/




    6、当编译成功之后，我们会得到一个提示，在

    /usr/local/php/lib/php/extensions/no-debug-non-zts-20090626/ 目录下有一个redis.so

    7、修改php.ini文件，让其加载redis.so文件，我们就可以操作redis

     php.ini在/usr/local/php/etc/php.ini目录下添加，如果有的话将前面的分号去掉即可

    Extendsion_dir="/usr/local/php/lib/php/extensions/no-debug-non-zts-20090626/"

    Extension="redis.so"


    8、重启服务nginx或者httpd
