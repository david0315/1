    • Mac OS 自带apache 启动不了的问题
    原文出处:本文由博客园博主早上六点半遇见五月天提供。
    原文连接:https://www.cnblogs.com/pawn-i/p/10873868.html

    问题：Mac OS 自带apache，启动不了，拒绝访问。

    1.先看一下错误日志

    /etc/apache2/httpd.conf

     

    2.然后我去找了一下/private/var/log/apache2/error_log。然后发现log文件夹下没有apache2这个文件夹。

    我想起来昨天清除了一些系统缓存，sudo du -sh * 发现/private 占了50多G的空间，所以，就把/private/var/log和/private/var/tmp两个文件夹下的东西全删除了……

    3.因为没法看日志，可以执行下面的命令

    sudo apachectl -k start 

     

    结果显示，因为log下不存在apache2这个文件夹，所以启动不了……

    所以解决方法就是：自己在log下新建一个apahce2这个文件夹，然后重启，就OK了
