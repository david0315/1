    定义和用法

    intdiv - 对除法结果取整

    版本支持

    PHP4PHP5PHP7
    不支持 不支持 支持

    语法

    intdiv ( int $dividend , int $divisor )

    intdiv() 返回 dividend 除以 divisor 商数的整数部分。

    参数

    参数必需的描述
    dividend 是 被除数
    divisor 是 除数。

     

    返回值

    dividend 除以 divisor 的商，对该商取整。

    示例

    var_dump(intdiv(3, 2));
var_dump(intdiv(-3, 2));
var_dump(intdiv(3, -2));
var_dump(intdiv(-3, -2));
var_dump(intdiv(PHP_INT_MAX, PHP_INT_MAX));
var_dump(intdiv(PHP_INT_MIN, PHP_INT_MIN));
var_dump(intdiv(PHP_INT_MIN, -1));
var_dump(intdiv(1, 0));
