    • PHP 将字符串中的数字转化为数组
    原文出处:本文由博客园博主小翼荨荨提供。
    原文连接:https://www.cnblogs.com/caia-agul/p/11883554.html

    $str ='现在是2019年11月18日下午17点25分';
    $result='';
    $arr=[];
    for($i=0;$i<strlen($str);$i++){
        if(is_numeric($str[$i])){
            $result.=$str[$i];
        }else{
            if($result!="") {
                $arr[] = $result;
                $result="";
            }
        }
    }


    打印数据：

    　　Array
    　　(
    　　　　[0] => 2019
    　　　　[1] => 11
    　　　　[2] => 18
    　　　　[3] => 17
    　　　　[4] => 25
    　　)
