    方式一：Windows版本通过composer来下载安装laravel框架

    一：laravel是php的一个web框架。laravel框架安装主要依赖composer工具，本经验就介绍一下怎么在windows上安装laravel框架。

    1、安装php web集成环境phpstudy2016，phpstudy2016支持PHP（5.2/5.3/5.4/5.5/5.6/7.0）版本自由切换。若安装其他的php web环境，如wmap等，就无需安装phpstudy2016。

    2、检查php版本，PHP版本 >= 5.6.4。其次要进入php.ini中看一下extension=php_openssl.dll是否是打开的，这个很重要，不然后续的操作就无法进行。

    3、安装composer工具，此工具很重要，包括安装laravel安装器或者通过composer create-project来创建laravel项目都要是使用composer，所以laravel框架安装主要依赖composer工具。

    二：使用laravel安装器安装

    1、先安装laravel安装器。安装命令：composer global require "laravel/installer"。

    2、安装好之后，在cmd命令行中，输入laravel命令，会输出该命令的版本和参数信息。

    3、在php web环境中创建laravel应用目录，目录文件名可以自定义。

    4、进入D:\phpstudy\WWW\laravel，进行安装。安装命令：laravel new test。test是应用名称，可自定义。

    5、测试安装是否成功。在浏览器地址栏中输入：http://localhost/laravel/test/public/index.php

    方式二：Laravel学习过程之Laravel安装及配置

    1. laravel环境

    Windows10

    Phpstudy

    Php7+apache2.4+mysql5.7

    Lavarel5.2

    2. 下载安装phpstudy
    1) 官网下载phpstudy20161103版本，或者百度提供的软件；

    官网下载地址：

    http://www.phpstudy.net/phpstudy/phpStudy20161103.zip

    2) 解压安装；

    双击解压后的exe文件，选择合适的目录进行安装；

    注意：选择安装的文件夹路径必须为非中文目录，防止安装或使用出错。

    3)  弹出“防止重复初始化”对话框，按照提示点击“是”；

    4) 启动phpstudy的apache和mysql服务；

    5) 切换php+apache版本，并重启成功；

    6) 访问localhost，默认是80端口；

    7）访问phpinfo.php文件显示正常，至此，phpstudy环境一键安装完成。

    8) 这是一个集成的wamp环境，有兴趣的童鞋可以试着自己搭建wamp或者lamp环境，难度挺大，适合练手提高环境配置能力和结局问题的能力。

    3. 下载安装laravel
    1) 安装PHP依赖包管理器composer

    a) 下载windows版本的composer

    官网地址：https://getcomposer.org/download/

    a) 安装composer

    如果没有安装过composer，直接双击下载好的composer的exe文件，按照正常“下一步”步骤直到安装完成即可。

    c) 注意事项：安装过程中过程中必须能够连接外网（翻墙）；

    d) 命令行窗口输入“composer”，显示如下，说明安装成功；

    e) 开启PHP拓展，在php.ini中；

    ;extension=php_openssl.dll

    ;extension=php_mbstring.dll

    ;extension=php_pdo_mysql.dll

    将前面的分号去掉，如果没有分号，则保持不变即可。

    2) 安装laravel

    Laravel包下载地址：

    http://down.golaravel.com/laravel/laravel-master.zip

    解压到你项目根目录F:\phpstudy\WWW\laravel-master

    注意：解压位置根据你自己安装的集成环境位置而定，phpstudy和wampserver都在根目录的www目录下。

    3) 打开cmd => f:

    cd phpstudy\WWW\laravel-master

    先将F:/ phpstudy\WWW\laravel-master /.env.example 修改成.env

    Cmd下执行命令

    composer install

    4) 执行命令，重新生成密匙

    php artisan key:generate

     

    最后一个步骤是针对Laravel5.x 以上。

    查看密匙是否生成成功可以去F:/WorkSpace/laravel-master/.env

    5) 在浏览器中输入localhost/laravel-master/public 将可以显示安装成功。

    6) 参考文章：

    http://blog.csdn.net/hxm_ss5224/article/details/51274372

    http://www.jianshu.com/p/2836017b5348

    7) 开启rewrite和vhost,在httpd.conf中找到

    #LoadModule rewrite_module modules/mod_rewrite.so

    #LoadModule vhost_alias_module modules/mod_vhost_alias.so

    #Include conf/extra/httpd-vhosts.conf

    将前面的#去掉，如果没有#，保持不变即可。

    8) 找到httpd.conf里的DocumentRoot "F:\phpstudy\WWW"附近的<Directory>标签对，将内容改成如下:

    <Directory />

     AllowOverride all

     Require all granted

    </Directory>

    9) 进入F:\phpstudy\Apache\conf\extra目录内的httpd-vhost.conf，再最后添加

    <VirtualHost *:80>

      ##ServerAdmin webmaster@ laravel.ding

      DocumentRoot "F:\phpstudy\WWW\laravel-master "

      ServerName laravel.ding

      ##ErrorLog "logs/laravel.ding-error.log"

      ##CustomLog "logs/ laravel.ding -access.log"common

    </VirtualHost>

    注意：路径文件夹要用加“/”，否则重启apache会失败。

    10) 进入C:\Windows\System32\drivers\etc目录内的hosts文件，在后面加上

    127.0.0.1    laravel.ding

    11) 访问laravel.ding/public出现laravel5的界面，说明配置成功.

    至此，第二种方式安装的laravel框架，从环境安装到lavarel在composer安装配置，再到本地访问站点配置已经全部完成，限免就可以尽情的使用laravel框架进行PHP编程了。
