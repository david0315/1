    • TP5 try{}catch{}异常捕获不到 解决办法
    原文出处:本文由博客园博主早上六点半遇见五月天提供。
    原文连接:https://www.cnblogs.com/pawn-i/p/11738460.html

    问题：TP5用下面的方法， 想要获取抛出的异常捕获不到……

    错误的写法：

    try{
……
} catch (Exception $e) {
    echo $e->getMessage();
}

    正确的写法：

    try{
……
} catch (\Exception $e) {
    echo $e->getMessage();
}
