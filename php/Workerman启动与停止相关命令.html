    start.php为入口文件

    一、启动

    1、以debug（调试）方式启动

    php start.php start

    2、以daemon（守护进程）方式启动

    php start.php start -d

    二、停止

    php start.php stop

    三、重启

    php start.php restart

    四、平滑重启

    php start.php reload

    五、查看状态

    php start.php status

    六、查看连接状态(需要Workerman版本>=3.5.0)

    php start.php connections
