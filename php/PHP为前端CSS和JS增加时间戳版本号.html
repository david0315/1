    • PHP为前端CSS和JS增加时间戳版本号
    原文出处:本文由博客园博主dreams7433提供。
    原文连接:https://www.cnblogs.com/phperlinxinlan/p/10783099.html

    一、PHP代码如下:

    function addVersion($url){
    $version = date("Y-m-d H:i:s",filemtime($_SERVER['DOCUMENT_ROOT'].$url)); 
    echo $url.'?v='.$version;
}

    二、HTML代码如下:

    <link href="<?php addVersion('/css/a.css');?>" rel="stylesheet" type="text/css">
<script src="<?php addVersion('/js/a.js');?>"></script>
